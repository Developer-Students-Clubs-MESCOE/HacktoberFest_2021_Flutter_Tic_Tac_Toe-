## Run it yourself
  
  ##### Installation
  - You need to have Flutter installed on your machine.
  - you need a IDE of your choice (VScode, Android Studio,etc...)
  
  ##### Running the Project
  - clone this repository to your machine and open it on your IDE.
  - finally run the project.
